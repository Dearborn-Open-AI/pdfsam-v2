Overwrite the file pdfsam-langpack.jar in your pdfsam_installation_path/lib directory.
To change pdfsam language open the config.xml with a text editor like Notepad++ (it seems UltraEdit gives some problem),
Change the value of the "i18n" tag with one of the language supported.

Languages supported
en_GB
it_IT
es_ES
pt_BR
ru_RU
sv_SE
de_DE
